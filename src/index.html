Administration MenuApi

Administration MenuApi is a Minecraft 1.21 plugin that provides a GUI for OP users to manage players, world settings, events, and system tools. It includes real-time stats, backups, reboots, weather control, and resource monitoring. Built with Bukkit/Paper, Kyori Adventure, Java 21, and Maven, it follows modern development standards.

  • Java
  • Spigot/Bukkit API

Plugin ClimaVote

This plugin allows players to vote for the weather type on the server. It counts the votes for each weather type and changes the weather if a minimum number of votes is reached. It also informs players of the voting result and the current weather via a chat message.

Java
Spigot/Bukkit API

Automatically restart server (Windows)

The "restart.ps1" PowerShell script is designed to start a Minecraft server using Fabric, and automatically restart it if it detects a crash. It also handles voluntary shutdowns and restarts in a controlled manner.

Shell scripting

Plugin CommandTPR

This plugin allows players to easily manage teleports. With "/tpr", a player sends a teleport request to another player. The receiver can accept it with "/tpa" or reject it with "/tpc". This makes it easier to cooperate and move quickly in the game.

Java
Spigot/Bukkit API

Solar Panel

This is a 3D model of a solar panel, designed in Blockbench with a low poly style ideal for Minecraft or projects with voxel aesthetics. Its simple but detailed structure shows a solid base with a series of photovoltaic cells on top, achieving a compact and functional design.
